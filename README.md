# QQPoisk Movie Redirect

Chrome extension that adds a button on Kinopoisk movie/series pages and opens the same page path on `qqpoisk.ru`.

Расширение Chrome, которое добавляет кнопку на страницах фильмов и сериалов Kinopoisk и открывает ту же страницу на `qqpoisk.ru`.

## Install Locally

1. Open `chrome://extensions/`
2. Enable Developer mode
3. Click "Load unpacked"
4. Select this project folder

## Локальная установка

1. Откройте `chrome://extensions/`
2. Включите режим разработчика.
3. Нажмите «Загрузить распакованное расширение».
4. Выберите папку проекта.

## Repository Hygiene

- Keep extension icons in `assets/icons/` as PNG: `icon16.png`, `icon32.png`, `icon48.png`, `icon128.png`.
- Main content script path: `scripts/content.js`.
